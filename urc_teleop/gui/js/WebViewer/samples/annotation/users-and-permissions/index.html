Instructions

Use the buttons to switch between users and show/hide annotations created by others. Each user has different permission levels:

  • Admin: Fully allowed to create/reply/edit/delete any annotations
  • User: Allowed to create/reply any annotations, but only allowed to edit/delete his/her own annotations
  • Read only: Not allowed to create/reply/edit/delete any annotations
